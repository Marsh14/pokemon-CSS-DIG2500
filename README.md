# Choose your fighter

## Project Description
I made a hypothetical character select screen for a neon retro fighting game, I used the pokemon API for thew character since they were easy to put images in the website

## Intended User or Player
Intended user is for fighting game players who are trying to find a character they want to play

## User / Player Goal
The user should be able to look through all the characters and select the one they like the most

## Inspiration Interfaces
I got inspired by character select screens from Persona 4 arena ultimax, street fighter 6, and tekken 8.
The P4AU CSS had super cool visual and cxool UI that I took inspiration from
SF6 had a cool neon background which is what I decided to use as my main theme
Tkenne 8 had good visual heiarchy, so I took inspiration from that and followed it

## Design Choices
- Layout: I used a flexbox layout to create a centered grid for the character roster, keeping everything evenly spaced and in the middle.
- Color: Neon blue and purple
- Typography: Courier New
- Visual hierarchy: I made sure to have the availble characters in the middle so thats what the players focus on
- Feedback states: When hovering over characters the card expands and glows, the button for starting also glows up
- Responsive design: When on mobile, the character cards stack vertically and not horizontally to match the phone size
